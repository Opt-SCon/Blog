# 个人博客项目总结

## 项目概述

一个基于纯前端的个人博客系统，使用原生 JavaScript 实现，数据存储使用 localStorage，无需后端服务器。

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript (ES6+)
- localStorage 数据存储

## 项目结构
```
project/
├── index.html # 首页
├── articles.html # 文章列表页
├── article.html # 文章详情页
├── editor.html # 文章编辑页
├── admin.html # 后台管理页
├── assets/
│ ├── js/
│ │ ├── api.js # 数据管理
│ │ ├── admin.js # 后台管理逻辑
│ │ ├── article.js # 文章详情逻辑
│ │ ├── articles.js # 文章列表逻辑
│ │ ├── editor.js # 编辑器逻辑
│ │ └── index.js # 首页逻辑
│ └── css/
│ ├── common.css # 公共样式
│ ├── admin.css # 后台样式
│ ├── article.css # 文章详情样式
│ ├── articles.css # 文章列表样式
│ ├── editor.css # 编辑器样式
│ └── index.css # 首页样式
```
## 核心功能实现

### 1. 数据管理 (api.js)

- 使用单例模式实现数据管理器
- localStorage 实现数据持久化
- 实现文章的 CRUD 操作
- 实现评论功能
- 实现点赞功能
- 实现文章分类
- 实现搜索功能

### 2. 首页功能 (index.js)

- 文章列表展示
- 搜索功能
- 滚动动画效果
- 响应式设计

### 3. 文章列表页 (articles.js)

- 分类筛选
- 搜索功能
- 文章卡片展示
- 加载动画
- 错误处理

### 4. 文章详情页 (article.js)

- 文章内容展示
- 评论功能
- 点赞功能
- 阅读量统计
- 相关信息展示

### 5. 编辑器页面 (editor.js)

- 文章编辑
- 实时预览
- 分类选择
- 自动保存

### 6. 后台管理 (admin.js)

- 数据统计
- 文章管理
- 评论管理
- 分类管理
